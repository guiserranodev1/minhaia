# 🤖 Minha IA com Gemini + Flask

Este projeto é um chatbot simples feito com **Python**, **Flask** e a API do **Google Gemini**, com interface web estilo chat.

---

## 🚀 Demonstração

Você digita uma mensagem no chat e a IA responde em tempo real no navegador.

---

## 🧠 Tecnologias usadas

* Python 🐍
* Flask (backend)
* Google Generative AI (Gemini) 🤖
* HTML + CSS (frontend)

---

## 🔑 Configuração da API (IMPORTANTE)

Para o projeto funcionar, você precisa criar uma API Key do Gemini:

1. Acesse o site oficial do Gemini:
   [https://aistudio.google.com/app/apikey](https://aistudio.google.com/app/apikey)

2. Gere uma nova API Key

3. Crie um arquivo chamado `.env` na raiz do projeto

4. Dentro dele, adicione:

```env
API_GEMINI_KEY=sua_api_key_aqui
```

⚠️ Nunca compartilhe sua API key publicamente.
