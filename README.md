# 🤖 Minha IA com Gemini + Flask

Este projeto é um chatbot simples feito com **Python**, **Flask** e a API do **Google Gemini**, com interface web estilo chat.

---

## 🚀 Demonstração

Você digita uma mensagem no chat e a IA responde em tempo real no navegador.

---

## 🧠 Tecnologias usadas

- Python 🐍  
- Flask (backend)  
- Google Generative AI (Gemini) 🤖  
- HTML + CSS + JavaScript (frontend)
